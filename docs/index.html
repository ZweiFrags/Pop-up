Meet the Zwei-up

Introduction

The Zwei-up is lightweight, written on pure js, dead simple solution for all your popup needs

Instalation

  • Step 1. Download archive
  • Step 2. Extract the contents of the dist folder from the archive
  • Step 3. Read Usage

Usage

Click me to open in modal!
Add data-ppp attribute to the element you want to open in modal: 
<p data-ppp>Click me to open in modal!</p>
Zwei-up is also capable of creating pop-ups with images 
<img src="src" data-ppp>
This plug-in uses tag figure for pop-ups. Thanks to this, you can add captions to images 
<img src="src" data-ppp data-ppp-title="text">
The plugin can also create galleries, just add the data-ppp-gal attribute. Two elements with same value of data-pp-gal will make a gallery 
<img src="first_src" data-ppp data-ppp-gal="1"><img src="second_src" data-ppp data-ppp-gal="1"><img src="third_src" data-ppp data-ppp-gal="1">

You can also combine all of these attributes for all your needs.

When you work with ajax you will have to use other attributes. Data-href="ajax" is used to indicate that we require ajax for the task, and href="data.html" specifies the site where the element needs to be found, data-ajax-id already specifies the element we need.
You can also use it for images 
<button data-href="ajax" href="data.html" data-ajax-id="id">
There is also data-ppp-id. This data attribute opens an element on the page with the same id value when clicked 
<button data-ppp-id="theid">



For form we added data-ppp-form. Forms opened in this way save all changes made by the user 
<form data-ppp data-ppp-form>
Forms can also be opened using a button or any other clickable elements thanks to data-ppp-id
